Back of the Napkin Video on APIs

The purpose of this assignment was to create a back of the napkin video that explain an engineering topic of my choice. I decided to create a video where I explained what web APIs are and some of their applications. I thought that I would be very uncomfortable with drawing and explaining my topic, but surprisingly I really enjoyed creating this video. I found that explaining what a web API is was much easier when I was able to create exactly what I was imagining, rather than finding an image that was only slightly similar. Being able to create my own drawings has also allowed me to utilize different metaphors to explain complex ideas about APIs, making it easier for the viewer to digest the information. Overall, this assignment has taught me to be more creative when it comes to communicating my ideas and that I should think outside the box more often to present information.
